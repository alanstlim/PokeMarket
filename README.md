## Sobre esse projeto
* PokeMarket é uma loja virtual criados usando a [PokeApi] (https://pokeapi.co/), seguindo alguns desafios propostos, afim de por em prática novos conhecimentos.
*  Esse é um projeto de código aberto, sinta-se livre para modificar da forma que desejar.

A ideia do aplicativo é:

>"3 e-Commerce que reutilizam um mesmo layout mudando apenas o tema de acordo com o elemento selecionado."

## Funcionalidades
- [x] Tema de acordo com o elemento selecionado.

- [x] Modal com informações extras do produto selecionado.
  
- [x] Carrinho usando hooks.

- [x] Perfil com histórico de compras e controle de recompensas por compra.

## Instalar

Basta clonar o projeto, abrir no VSCode (Ou editor de sua preferência)

Usar o comando: yarn install (Para instalar as dependências necessárias)

Usar o comando: yarn start (Para iniciar o projeto)
